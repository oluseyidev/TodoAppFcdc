# Odin Todo App

Instructions to run the app locally:

1. Setup backend:
   - Go to `api/`
   - Copy `.env.example` to `.env` and fill in your MongoDB URI
   - Run `npm install` then `npm run dev`

2. Setup frontend:
   - Go to `web/`
   - Copy `.env.example` to `.env` and set `VITE_API_URL`
   - Run `npm install` then `npm run dev`

3. Deploy:
   - Frontend to Vercel
   - Backend to Render
